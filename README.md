# tw3-dao

This is the organizational repo for TW3 (TechnicallyWeb3). Everything someone needs to know about TW3's values, governance, contribution standards, and how projects earn community support starts here.

**Start with [PHILOSOPHY.md](PHILOSOPHY.md)** — it is the source of truth for what TW3 believes, why we exist, and what we require of projects that seek community investment.
